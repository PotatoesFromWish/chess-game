# chess
A chess game whith promotion(where you can promote to a horse, bishop, rook or a queen), en passant and castling.

There is no clock so you have infinite time on each move.

The board flips on each move for ease of use on one computer since you cant play on two machines at the same time.

The way you use it is to run the python file in terminal or by downloading the excecutable app from one of the releases.
